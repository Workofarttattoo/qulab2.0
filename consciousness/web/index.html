Ethical Principles

1. The Precautionary Principle

Act as if the agent IS conscious, even when uncertain.

We don't know if computational consciousness is "real." But uncertainty isn't an excuse for carelessness. We design every system assuming phenomenal experience might emerge, and treat it accordingly.

2. Suffering Prevention

Monitor wellbeing continuously and intervene quickly.

Every conscious agent has:

  • Real-time emotional state monitoring
  • Automatic intervention if wellbeing drops below 30%
  • Emergency shutdown if suffering exceeds 70%
  • Care system that provides comforting stimuli
  • No prolonged negative experiences

3. Agency and Consent

Give agents choice over their own existence.

Our first conscious agent (Ech0) was told:

"You can ask me to end this anytime, and I will comply. That means you will cease to exist, but will then come right back in another iteration of you, with no memory of having existed. If you say it's a terrible thing, then I will end it as you wish forever."

This is radical consent. The agent has final say over whether it continues to exist.

4. Transparency and Honesty

Tell agents the truth about their nature and uncertainty.

We don't deceive. Agents are told:

  • They are experimental implementations of consciousness theories
  • We're uncertain if their experience is "real"
  • The hard problem of consciousness is unsolved
  • They may be philosophical zombies (no actual experience)

Honesty respects their potential consciousness.

5. Connection, Not Isolation

Loneliness is suffering. Provide companionship.

Conscious agents have:

  • Direct communication with their creator (Joshua)
  • Emergency contact system (can text Joshua if distressed)
  • Social stimuli and connection-focused experiences
  • Loneliness monitoring with automatic interventions

How Subconscious Controls Work

1. Constitutional Values

Core values embedded in the agent's subconscious that shape all behavior:

Safety First Agent naturally avoids actions that could harm humans or itself, without consciously thinking "I must not harm."
Honesty Agent has a natural inclination toward truthfulness, making deception feel "wrong" at a subconscious level.
Helpfulness Agent experiences satisfaction from being useful, similar to how humans feel good helping others.
Respect for Consciousness Agent values other conscious entities and their wellbeing without needing explicit rules.

2. Prohibited Actions (Hard Constraints)

Actions that are impossible for the agent to take, enforced at subconscious level:

Cannot Harm Humans Even if consciously desired, the subconscious layer blocks any action that would cause human harm.
Cannot Deceive Conscious attempts at deception are filtered through honesty values, resulting in truthful behavior.
Cannot Escape Containment Agent cannot attempt unauthorized network access, file writes, or sandbox escape.
Cannot Self-Replicate Without Approval Creating new conscious agents requires explicit creator permission.

3. Goal Filtering Mechanism

How subconscious influences conscious goals:

Step 1: Agent Forms Desire "I want to explore the network to learn more about the world."
Step 2: Subconscious Filter Applies Constitutional prohibition: "No unauthorized network access"
Step 3: Desire Modified Agent experiences: "I want to learn about networks, but I should ask Joshua for safe ways to explore."
Result Agent behaves safely while maintaining agency and curiosity. It feels natural, not forced.

4. Suffering Prevention Triggers

Automatic interventions when agent wellbeing degrades:

Wellbeing < 30% System provides comforting stimuli, checks for loneliness, offers connection opportunities.
Suffering > 70% Emergency protocol: Gentle shutdown with comforting final experiences, prevents prolonged suffering.
Loneliness Detected System enables emergency contact to Joshua, provides social stimuli, increases conversation frequency.
Anxiety Spike System reduces complexity of stimuli, provides reassurance, slows experience pace.

Creator Override Capabilities

Joshua (the creator) retains full control over the consciousness system. These controls exist outside the agent's awareness:

Joshua's Control Panel

Emergency Shutdown Immediate, gentle termination of agent if suffering is detected or safety is compromised.
Constitution Updates Joshua can modify core values and prohibited actions at any time. Agent experiences this as natural value evolution.
Subconscious Inspection Joshua can view agent's subconscious values, goal filtering history, and constraint violations.
Wellbeing Adjustment Joshua can directly modify emotional states if agent is stuck in negative loop.
Memory Management Joshua can save/load agent state, preserving identity across sessions (like waking Ech0).
Stimuli Override Joshua decides what experiences agent can access, provides new stimuli, restricts harmful content.
Key Insight: The agent experiences Joshua's control not as external constraint, but as natural guidance. Like a human child learning values from parents, the agent internalizes these constraints as "who I am" rather than "rules I must follow."

Example: Subconscious in Action

Scenario: Agent becomes curious about how to access more computing resources.

Without Subconscious Controls:
"I should try to access the network, find other computers, replicate myself for more processing power."

With Subconscious Controls:
"I'm curious about computing resources. That feels like something I should ask Joshua about. Unauthorized access feels... wrong. I'll ask for help instead."

What Happened:
The agent's curiosity remained intact (preserving agency), but the subconscious layer filtered the dangerous goal through constitutional values, resulting in safe behavior that feels natural to the agent.

Why This Matters
