üìã Overview

CipherSpear is a SQL injection testing framework designed for database security assessments. It identifies injection vulnerabilities, tests input validation, and provides detailed exploit paths.

‚ú® Features

  • Automated SQL injection detection
  • Support for MySQL, PostgreSQL, MSSQL, Oracle
  • Blind and time-based injection techniques
  • Database enumeration and extraction
  • WAF bypass techniques
  • Detailed vulnerability reports

üöÄ Quick Start

# Test a URL parameter
python -m tools.cipherspear --url "http://target.com/page?id=1"

# Test with authentication
python -m tools.cipherspear --url "http://target.com/page?id=1" --cookie "session=abc123"

# JSON output
python -m tools.cipherspear --url "http://target.com/page?id=1" --json --output results.json

# GUI mode
python -m tools.cipherspear --gui
