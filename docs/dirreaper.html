üìã Overview

DirReaper discovers hidden web resources, directories, and files through intelligent brute-forcing and wordlist-based enumeration. Essential for web application assessments.

‚ú® Features

  • High-speed multi-threaded scanning
  • Custom wordlist support
  • Recursive directory discovery
  • Status code filtering
  • Pattern matching and exclusions
  • Progress tracking and resume capability

üöÄ Quick Start

# Basic directory scan
python -m tools.dirreaper --url http://target.com

# Custom wordlist
python -m tools.dirreaper --url http://target.com --wordlist /path/to/wordlist.txt

# Recursive scan with depth limit
python -m tools.dirreaper --url http://target.com --recursive --depth 3

# GUI mode
python -m tools.dirreaper --gui
