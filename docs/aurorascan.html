üìã Overview

AuroraScan is a professional network reconnaissance tool designed for security assessments. It discovers open services, identifies potential attack vectors, and provides detailed host information.

‚ú® Features

  • Fast multi-threaded port scanning
  • Service version detection
  • OS fingerprinting
  • Network topology mapping
  • JSON output for automation
  • Customizable scan profiles (stealth, normal, aggressive)

üöÄ Quick Start

# Basic scan
python -m tools.aurorascan 192.168.1.0/24

# Stealth scan with version detection
python -m tools.aurorascan 192.168.1.0/24 --profile stealth --version-detect

# JSON output for automation
python -m tools.aurorascan 192.168.1.100 --json --output scan-results.json

# GUI mode
python -m tools.aurorascan --gui

‚öôÔ∏è Parameters

  • target - IP address or CIDR range to scan
  • --profile - Scan profile: stealth, normal, aggressive
  • --ports - Port range (default: 1-1000)
  • --timeout - Connection timeout in seconds
  • --json - Output results in JSON format
  • --gui - Launch graphical interface
