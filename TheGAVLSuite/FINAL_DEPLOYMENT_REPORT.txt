═══════════════════════════════════════════════════════════════════════════════
                        QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

To Deploy TheGAVL:

1. Clone repository:
   git clone https://github.com/Workofarttattoo/AioS.git
   cd TheGAVLSuite

2. Create configuration:
   cp .env.example .env
   # Edit .env with your database password & API key

3. Start services:
   docker-compose up -d

4. Verify deployment:
   curl http://localhost:8000/api/v1/health

5. Make test prediction:
   curl -X POST http://localhost:8000/api/v1/predict \
     -H "X-API-Key: your-api-key" \
     -d '{
       "case_name": "Test v. State",
       "case_id": "test-001",
       "court": "scotus",
       "opinion_text": "...",
       "issue_area": "criminal"
     }'

6. Send outreach emails:
   Review OUTREACH_EMAILS.md
   Personalize templates
   Send to 100+ target leads
