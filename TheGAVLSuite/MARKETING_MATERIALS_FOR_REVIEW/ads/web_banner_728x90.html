Export Instructions

To use these banners:

  1. Open this HTML file in Chrome or Firefox
  2. Use browser screenshot tool or print to PDF
  3. Crop each banner to exactly 728x90 pixels
  4. Save as PNG or JPEG (PNG recommended for web)
  5. Upload to ad platform or website

For professional design:

  • Send this file to a designer to create high-resolution versions
  • Request 2x resolution (1456x180) for retina displays
